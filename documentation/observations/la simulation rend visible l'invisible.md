# La simulation rend visible l'invisible 

> [...] le virtuel désigne uniquement la capacité des appareils numériques à interface graphhiques de produire des réalités informatiquement simulées, que ce soit des environnements logiciel (comme Windows ou Mac OS) ou de véritable mondes virtuels (comme Second Life)." Le virtuel est donc considéré "[...] comme le seul moyen pour le noumène numérique de devenir une réalité phénoménale, [...] le seul moyen de l'appréhender ou de le manifester. C'est peut-être pour cela que le virtuel n'est rien d'autre que du simmulationnel : ==pour rendre visible une réalité invisible, rien de tel que de la simuler==, afin d'en créer une instance phénoménale.
