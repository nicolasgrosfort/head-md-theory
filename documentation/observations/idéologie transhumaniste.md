# Idéologie transhumaniste

… cette accélération technologique adoptent les schémas narratifs propres à l'idéologie transhumaniste, qui se déploie dans la Silicon Valley depuis plus d'une trentaine d'années. ==Celle-ci s'appuie sur un scénario tout tracé, selon lequel l'humanité, dépassée par les performances de ses machines, sera bientôt à la merci d'une "superintelligence" prenant le contrôle de son destin==. Couramment désigné sous le nom de "singularité technologique" ce point de bifurcation se voit tour à tour revendiqué ou dénoncé, chacun s'accordant néanmoins sur son arrivée imminente.
