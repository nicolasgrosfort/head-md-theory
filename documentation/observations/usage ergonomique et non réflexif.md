# Usage ergonomique et non réflexif

Les citoyens ont rarement les moyens de comprendre les technologies qu'ils utilisent  quotidiennement, dont les fonctionnalités ont été conçues pour les empêcher de penser. Au contraire, ==les normes techniques, toujours porteuses de projets politiques, sont dissimulées derrière des interfaces ergonomiques, qui permettent un usage immédiat et intuitif, mais rarement réflexif==.
