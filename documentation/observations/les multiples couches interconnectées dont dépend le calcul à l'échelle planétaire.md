# Les multiples couches interconnectées dont dépend le calcul à l'échelle planétaire

La pile est un concept utilisé dans les études scientifiques et technologiques, la philosophie de la technologie et les études sur les médias pour décrire les multiples couches interconnectées dont dépend le calcul à l'échelle planétaire. Le terme a été introduit par Benjamin H. Bratton dans un essai de 2014 et développé dans son livre de 2016 The Stack : On Software and Sovereignty.
